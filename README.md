## 🧠 冷知識一則
在1919年，法國發明家喬治·梅爾斯（Georges Méliès）製作了一部名為《月球旅行記》的電影，其中描繪了宇航員乘坐炮彈到月球的故事。這部電影被視為科幻電影的先驅，同時也啟發了許多後來的太空探索作品。值得注意的是，梅爾斯應用了早期的特殊效果技術，為電影史帶來了革命性的變化，然而，由於當時的電影和電影技術未被重視，這部作品在隨後的十年內幾乎被遺忘，直到後來才重新受到讚譽。
