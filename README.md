## 🧠 冷知識一則
在19世紀，某些法律規定人們可以通過“半途而廢”來解決婚姻問題。這意味著，如果一位配偶在結婚後的頭六個月內沒有發生性關係，另一方可以要求解除婚姻，這被認為是一種合法的理由。這項法律在某些地區仍然存在，反映了當時社會對婚姻的看法和期待。
