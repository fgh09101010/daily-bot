## 🧠 冷知識一則
在1969年，人類首次登陸月球時，宇航員來到月球的時候帶了一種特別的東西——一包月球的土壤和岩石樣本，這些樣本在地球的實驗室中進行了分析。然而，由於安全考量，NASA在返回地球後將樣本封存了幾十年，直到1993年才將它們重新開放給科學家進行研究，這些樣本至今仍然提供著對月球形成和歷史的重要見解。
