## 🧠 冷知識一則
當你打噴嚏的時候，通常會瞬間停止你的心臟跳動。這是因為在噴嚏的過程中，胸腔壓力增加，暫時壓迫心臟，導致心臟的跳動暫時減弱。不過，這種情況是非常短暫的，幾乎不會影響健康，但卻是一個有趣的人體反應！
