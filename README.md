## 🧠 冷知識一則
你知道嗎？章魚有三顆心臟！其中兩顆心臟負責將血液泵送到鰓中，以便獲取氧氣，而第三顆心臟則將氧氣豐富的血液泵送到身體的其他部位。在章魚游泳時，第三顆心臟會暫時停止跳動，這就是為什麼章魚更喜歡爬行而不是游泳的原因。
