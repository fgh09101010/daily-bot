## 🧠 冷知識一則
你知道嗎？大多數的橙色橘子其實是綠色的！在熱帶地區，由於氣候的影響，橘子的表皮會保持綠色，即使它們已經成熟並且味道甜美。在這些地區，人們多數會選擇綠色的橘子，而不是傳統的橙色橘子，因為橘子顏色並不代表其內部的成熟度或味道。
