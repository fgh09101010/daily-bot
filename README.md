## 🧠 冷知識一則
你知道嗎？香蕉實際上是漿果，而草莓卻不是！根據植物學的定義，漿果是由一個花的子房發育而來，並且通常含有多顆種子。香蕉符合這個定義，而草莓則是由多個雌蕊發育而來的聚合果，因此不算是真正的漿果。這真是令人意外的果實分類！
