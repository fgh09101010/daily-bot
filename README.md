## 🧠 冷知識一則
你知道嗎？大多數的橄欖油在玻璃瓶中保存時，會比塑料瓶中的更長久，因為光和空氣會影響油的品質。橄欖油中富含的抗氧化劑能阻止油變質，但光照會減少這些抗氧化劑的效果。因此，很多高品質的橄欖油都會選擇使用深色玻璃瓶來儲存，這樣可以幫助保持油的新鮮和味道。
