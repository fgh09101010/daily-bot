## 🧠 冷知識一則
你知道嗎？海星並不是真正的魚。雖然名字中有「星」和「魚」，但海星屬於棘皮動物，這意味著它們和海膽、海參等動物是近親。海星擁有獨特的再生能力，能夠再生失去的腕或甚至整個身體！這讓它們在海洋生態系統中扮演了重要的角色。
