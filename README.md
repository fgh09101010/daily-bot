## 🧠 冷知識一則
你知道嗎，章魚擁有三顆心臟？其中兩顆心臟負責將血液泵送到鰓，以獲取氧氣；而第三顆心臟則將氧氣充足的血液送到身體的其餘部分。當章魚游泳時，第三顆心臟會暫時停止跳動，這也是為什麼章魚更喜歡爬行而不是游泳，因為游泳會讓它們更加疲憊。
