## 🧠 冷知識一則
你知道嗎？章魚有三顆心臟！兩顆心臟用來將血液泵送到鰓中，這樣它們就能獲取氧氣；第三顆心臟則將氧氣豐富的血液泵送到全身。當章魚游泳時，第三顆心臟會停止跳動，這就是為什麼它們更喜歡爬行而不是游泳的原因，因為游泳會讓它們感到疲憊。
