## 🧠 冷知識一則
你知道嗎？香蕉其實是漿果，而草莓卻不是！在植物學上，漿果是由一個單一的子房發育而來的果實，通常包含許多種子。香蕉符合這個定義，而草莓則是由多個子房組成的聚合果，因此不算真正的漿果。這個反轉的定義真的讓人驚訝！
