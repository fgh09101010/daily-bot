## 🧠 冷知識一則
你知道嗎？章魚有三顆心臟！其中兩顆心臟負責將血液泵送到鰓，以便吸收氧氣，而第三顆則負責將氧氣充足的血液泵送到全身的其他部位。當章魚游泳時，第三顆心臟會暫時停止運作，因此它們通常更喜歡爬行而不是游泳，以節省能量。
