## 🧠 冷知識一則
你知道嗎？章魚有三顆心臟！其中兩顆負責將血液送往鰓，讓它們獲取氧氣，而第三顆則將氧氣豐富的血液送往全身。當章魚游泳時，它的第三顆心臟會停止跳動，這也是為什麼章魚通常更喜歡爬行而不是游泳，因為游泳會讓它們感到疲憊。
